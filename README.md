# IoT-Based Smart Medical Kit for Paralyzed Patients

## 📌 Overview
This project is an IoT-based healthcare system designed to assist paralyzed patients by continuously monitoring vital health parameters and enabling gesture-based communication with caregivers. The system provides real-time monitoring, cloud connectivity, and emergency alerts to improve patient safety and quality of care.

## ✨ Features
- 🤚 Gesture-based communication using Flex Sensors
- 🌡️ Body Temperature Monitoring (LM35)
- ❤️ Heart Rate & SpO₂ Monitoring (MAX30100)
- 💧 IV Drip Level Monitoring (Load Cell + HX711)
- ☁️ Real-time cloud monitoring
- 🚨 Automatic emergency alerts
- 📟 LCD display for live health data

## 🛠️ Hardware Components
- NodeMCU (ESP8266)
- LM35 Temperature Sensor
- MAX30100 Pulse Oximeter
- Flex Sensors
- HX711 Load Cell Amplifier
- Load Cell
- LCD Display
- Buzzer

## 💻 Software & Tools
- Arduino IDE
- Embedded C
- ESP8266 Libraries

## ⚙️ Working Principle
The system collects patient health data using multiple sensors. The NodeMCU processes the sensor values and uploads them to the cloud for remote monitoring. Flex sensors detect hand gestures to communicate patient needs, while abnormal health conditions trigger automatic alerts for caregivers.
